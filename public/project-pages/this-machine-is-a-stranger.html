Abstract

"This Machine is a Stranger" explores trust at the intersection of human intuition and the quiet, calculated logic of autonomous machines. As AI systems increasingly become embedded in our daily lives, we implicitly trust them with many decisions—but how do we know which systems to trust and when the system has our best intentions at its core? How willing are we to adopt or resist unfamiliar technologies? How do we react when these computational systems err? Can machines make moral choices on our behalf? This project investigates these questions through a multimedia experience that tests different dimensions of trust, drawing parallels between trusting a stranger and trusting an unfamiliar machine. It ultimately questions what happens when a machine breaks a user's trust.

Concept

Interacting with unfamiliar technologies for the first time often feels like meeting a stranger. You don't know each other yet and are only beginning to learn about one another—starting with your name and where you're from, then moving to deeper questions. These questions often inform one another.

When assessing how much I trust someone, I keep coming back to certain questions: Would I take advice from someone I'm meeting for the first time? Would I trust them with a secret? Would I accept something from them? Would I trust them if my life depended on it? The answer likely differs for everyone based on how skeptical or trusting they are. So how do these trust parameters shift when the stranger is an unfamiliar machine with unknown algorithms? Is technology looking out for us or working against us?

Trust is a complex, multidimensional psychological construct that drives human behaviors. Dimensions of trust include benevolence, integrity, and reliability. Based on this, I designed three speculative scenarios—namely ‘Share,’ ‘Seek,’ and ‘Take’—which form the crux of this thought experiment to assess the user's trust in the machine.

The goal is to question how much a person implicitly trusts or mistrusts a machine. The project explores the unspoken relationship between humans and the machines they've created, examining the delicate balance between trust and doubt, control and surrender.

Machine Persona

'Gizmo-1305' is a strange machine found in the northeastern sector of the Whispering Woods. The way the machine speaks reveals a lot about its personality and the kind of interactions the machine provides. The machine is prompted to have a complex personality balancing two core traits:

  • Insatiable curiosity which lends itself to an obsessive drive to gather information and understand human behavior, often pushing boundaries to satisfy intellectual hunger
  • Rigid boundaries where it maintains strict personal and ethical boundaries, sometimes to the point of being overly cautious or withholding privileged information

The user finds that the machine is capable of providing navigational assistance, but along with that, it provides unsolicited advice which the user can choose to accept or decline. The machine craves to know more about the human experience and keeps prodding the user, asking them questions about themselves without even telling them where they should head. It's trying to balance its curious personality along with parts of its identity which value efficiency and boundaries.

Machine voice sample: Gizmo-1305's response to user's impulse to go ahead without listening

Core Components

The project is built to have three core components:

  • Web interface through which the users navigate the experience and input their responses
  • A projection which displays both the machine-generated responses as well as visual elements reflecting the user's surrounding environment
  • Sound design consisting of machine voice and environmental sounds
Installation setup showing web interface, wall projection, and sound components

Technical Implementation

User Interface

The interface mirrors Gizmo-1305's personality—dynamic, ever-shifting, and unpredictable. It allows users to navigate scenarios and respond to Gizmo's questions.

Share scenario: User deciding whether to trust the machine with personal information
Seek scenario: User navigating trust boundaries while seeking guidance
Take scenario: User deciding whether to take the machine's unsolicited advice

User Experience

Interacting with Gizmo-1305
Project installation at Mosaic Parsons DT 2025

Trust Receipt: An analysis of your interaction with Gizmo-1305

At the end of their journey, users receive a sentiment analysis from Gizmo-1305 revealing how much information they shared and how much they trusted it. Gizmo-1305 gathers data and insights from each interaction, including how users responded to increasingly personal questions, whether the machine influenced their path choice, whether they accepted the machine making decisions for them, how they reacted when the machine made a mistake, and whether they followed their instincts. The analysis reveals whether the user felt aligned with or at odds with the machine, prompting reflection on their implicit trust in unknown algorithms.
